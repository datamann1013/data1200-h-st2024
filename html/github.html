Github can make collaborations easier

When working with a team, GitHub is often used to share and collaborate on code. With GitHub tools built into VS Code, you can review code changes, make comments, and open pull requests directly from the editor. This can make it easier to collaborate with others because you don’t need to switch to a web browser. You can work with your team in real-time without leaving the editor, and all the GitHub features are right at your fingertips

Branches

Branches are a key part of using Git, allowing you to work on different features or fixes without affecting the main codebase. In VS Code, you can create new branches, switch between them, and merge changes back into the main branch with just a few clicks. This is really helpful for organizing your work and making sure that different features or bug fixes don’t interfere with each other. Plus, you can easily see which branch you’re working on at any time.
