Accessibility

welcome to accessibility hell

Font and Readability:

We chose the font Verdana, with sans-serif as a backup. The justification for this choice was based on it's legibility and accessible design. The font differatiates the letters well even in small textsizes.

All paragraphs use the reccommended 1.5 line-height to make the text easier to read. We've ensured that the textsize is rem to scale it properly based on individual user needs. letter-spacing is 0.04rem

Structure:

General layout, clear headers and high-resolution images. Understandable: Implicitly imply the content without needing to read everything. Keeping headers in correct order The content scales nicely when transitioning between smaller and larger screen sizes. Spesifically, the text's width and height remains the same by using media queries. Max width for larger screens???

Colors:

Choices and justification Lightmode and darkmode = contrast on both! darkmode links are lighter than lightmode links while still being assosiated to hyperlinks

Images

PNG, WebP, JPEG Backup image formatting nescessary? Multiple versions of one image to support different screen sizes

Alt text

We've made sure to include a descriptive alt text for visually impaired users. Images and icons should have this

Interactibility:

Screen readers (aria-label). Tabindex for non-mouse users
