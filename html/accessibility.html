We chose the font Verdana, with sans-serif as a backup. The justification for this choice was based on it's legibility and accessible design. The font differatiates the letters well even in small textsizes. We've avoided fonts featuring “imposter letter shapes”. These include characters that are visually similar, but not the same, for example uppercase Is, the number 1 and lowercase Ls.

All paragraphs use the reccommended 1.5 line-height to make the text easier to read. We've ensured that the textsize is rem to scale it properly based on individual user needs. letter-spacing is 0.04rem

General layout, clear headers and high-resolution images. Understandable: Implicitly imply the content without needing to read everything. Keeping headers in correct order The content scales nicely when transitioning between smaller and larger screen sizes. Specifically, the text's width and height remains the same by using media queries. Max width for larger screens???

Choices and justification Lightmode and darkmode = contrast on both! darkmode links are lighter than lightmode links while still being assosiated to hyperlinks
